# Three-Minute Demo Script

## 0:00–0:20
“Most recommendation apps know what is nearby. FREE Time knows what will realistically work for this person, in this exact window, today.”

## 0:20–0:45
Show Maria. Point out calm environment, pet access, reliable seating, and a two-hour return deadline. Choose Together, Inside, Pay.

## 0:45–1:15
Generate plans. Explain that deterministic code first rejects anything that violates time, pet, seating, crowd, noise, or walking constraints.

## 1:15–1:40
Open one result. Show fit score, uncertainty, and GPT-5.6 explanation.

## 1:40–2:00
Choose the plan. Show “And don’t forget…” with wallet, keys, weather clothing, pet items, departure time, and saved errand.

## 2:00–2:20
Report a chair issue. Regenerate and explain that the profile now treats reliable seating as a hard requirement.

## 2:20–2:40
Switch to James. Generate an active outdoor plan. Emphasize that age or demographics do not determine the result; functional reality does.

## 2:40–3:00
“Codex built and tested the application. GPT-5.6 handles nuanced whole-person explanations. Deterministic code enforces the rules. FREE Time helps you feel good about what you do next.”
