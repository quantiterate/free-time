# Claude Chrome Handoff

Use this checklist to complete account-bound submission work.

1. Create or open the Devpost project.
2. Upload/push this repository to a judge-accessible GitHub repository.
3. Add the OpenAI API key to the deployment environment.
4. Deploy the app to a public URL.
5. Verify the public app in a logged-out browser.
6. Run `npm test` and capture the successful output.
7. Record the three-minute video using `DEMO_SCRIPT.md`.
8. Upload the video publicly or unlisted as the contest permits.
9. Paste `DEVPOST_COPY.md` into the submission fields.
10. Add the repository URL, live demo URL, video URL, and required Codex `/feedback` session ID.
11. Confirm eligibility, ownership, permissions, and contest rules.
12. Submit before the deadline.

Do not claim that live venue, smartwatch, medical, or nationwide data is implemented when it is not. Present those as next steps.
